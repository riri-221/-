ROUGH CITY MECHANIC

ラフシティメカニックシート

FiveM の中ではなく、ブラウザ単体で動くローカルアプリです。 file で直接開けるのでオフラインでも使えます。 価格表はこのアプリ内で編集して保存、オーダーシートは貼り付けるだけで必要アイテムと請求候補を確認できます。 メモ欄も追加しているので、作業中の補足や下書きも同じ場所に残せます。

MODE 01

請求金額

対象にチェックを付けて数量と単価を見ながら合計を出します。 金額はその場で変更でき、ローカル保存と JSON 書き出しにも対応しています。

MODE 02

オーダーシート解析

jim-mechanic の mechboard からコピーした内容を貼り付けると、 カスタム項目と必要アイテムを一覧化し、請求候補にも反映できます。

MODE 03

半グレ、ギャングカラー

半グレとギャングのカラー指定を一覧で確認して、 jim-mechanic 名義の内容を見返したり新規追記できます。

MODE 04

メモ欄

作業メモ、連絡事項、請求前の下書きを残せるローカル保存ページです。 JSON の書き出しと取り込みにも対応しています。
